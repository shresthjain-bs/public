Text Contrast Benchmark

Every case the deterministic contrast solution must resolve. Each croppable target has a stable id and data-expect (PASS / FAIL / REVIEW), data-ratio (worst-pair WCAG ratio where computable), and data-note. Crop .target rects by id. Thresholds: 4.5:1 normal text, 3:1 for ≥24px. Manifest: window.TC_MANIFEST.
